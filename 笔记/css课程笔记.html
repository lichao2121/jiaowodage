外链式和导入式的区别

外链式和导入式的区别 面试题
link和@import虽然都是引入外部的css文件，但是他们是由天差地别的区别的
link是html标签，@import完全是css提供的方式，要写在css文件或者style标签中。
他们的加载顺序也是有区别的，当一个页面被加载的时候，link引用的css文件会被同时加载，而@import引入的css文件会等页面全部加载完后再加载。
当使用javascript控制DOM去改变css样式的时候，只能使用link标签，因为@import是不能被DOM控制的。

相对定位

**特点**
  • 不脱离文档流
  • 当位置发生改变时，原来的位置还被占用
  • 相对定位的层级大于文档流的其他元素（会盖在其他元素的上面）
  • 参照物是自己（本身）
  • 给绝对定位当做参照物来用
  • 当相对定位同时设置left right 值时 left生效；同时设置top，bottom值时top值生效。

在什么情况下使用相对定位
1.当自己本身位置发生改变时，又不影响其他元素，这时候可以用相对定位；
2给绝对定位当做参照物来用

绝对定位

1.当不设置参照物时，参照物是body 2.认为设置参照物时，必须满足两个条件：

1.必须是这个绝对定位的父元素（祖辈元素）
2.这个父元素必须有定位属性
3.脱离文档流
4.给元素只设置绝对定位，不设置四个方向值时，如果这个绝对定位前面有其他元素，会默认排在这个元素后面

  • 在不设置宽高时，宽高是由本身内容决定的
  • 给绝对定位元素设置宽高时（100%），宽高继承参照物的宽高
5.当绝对定位元素同时四个方向值时（left right top bottom）时，left 和top生效。

固定定位

**特点**
  • 参照物是浏览器的可视窗口
  • 在不设置宽高时，宽高是由内容决定的
  • 脱离文档流

CSS3动画

transition 过渡

  • 1.设置过渡效果的css属性的名称
    • all 全部
    • width
    • height
    • left
    • right
    • ….
  • 2.完成过渡效果需要多少秒
    • 5s .5s
  • 3.速度效果的速度曲线
    • linear 以相同速度开始至结束的过渡效果(线性过渡)
    • ease 慢速开始，然后变快，然后慢速结束的过渡效果(平滑过渡) —默认
    • …..
  • 4.过渡效果开始的延迟时间
    • .1s (以秒为单位)

transform 属性向元素应用2D、3D转换

translate 移动位置

  • translate(x,y) 定义2d转换
  • translate(x) 代表x轴转行
  • translate3d(x,y,z) 定义3d转换

scale 缩放(缩小、放大)

  • scale(n) 倍数 — 常用
  • scale(x,y) 定义2d缩放转换

rotate 旋转

给元素顺时针旋转一个角度(单位:deg)
只能设置一个值

  • rotate(30deg); 顺时针旋转
  • rotate(-30deg); 逆时针旋转

skew 倾斜

  • skew(30deg); 逆时针倾斜
  • skew(-30deg); 顺时针倾斜
  • skew(x,y) 沿着x和y轴的2d倾斜转换

一般常用skew(30deg);只有一个值时

animation 动画

规定好动画效果

  • 指定关键帧动画 @keyframes
  • 动画的名称是自定义(自己随便起名的)
  • 大括号{} 规定动画划分多少块
    • 0%{} 动画开始位置 100%{} 动画结束位置 —-特点 可以规定动画的效果，例如：0%{} - 50%{} 100%{}
    • from{} 动画开始位置 to{} 动画结束位置–只有动画开始和结束的效果
      • transform 变换：translate 坐标值、scale 缩放 、rotate 旋转、skew倾斜

给指定元素调用动画效果

  • 1) animation-name 动画的名称
  • 2) animation-duration 动画所需要完成的时间

  • 3) animation-timing-function 动画的速度曲线(运动方式)

    • linear 动画从头到尾的速度是相同的(线性过渡)
    • ease 动画以低速开始，然后加快，在结束前变慢。(默认值，平滑过渡)
    • …..

  • 4) animation-delay 动画延迟的时间

  • 5) animation-iteration-count 动画运行的次数
    • 具体的次数 n
    • infinite 无限次
  • 6) animation-fill-mode 动画结束保持状态(规定动画在播放之前或之后，其动画效果是否可见)
    • both(开始和结束，向前和向后填充模式都被应用) – 常用
    • backwards (开始)
    • forwards(结束)

both 保留动画关键帧中 开始的一帧(0%)或结束的一帧(100%)
backwards 保留动画关键帧中 开始的一帧(0%)
forwards 保留动画关键帧中 结束的一帧(100%)

  • steps

是一个timing function(定时功能)，允许我们将动画或者过渡分割成段，而不是从一种状态持续到另一种状态的过渡。
参数是一个正值，指定我们希望动画分割的段数。
